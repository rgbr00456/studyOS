⚙ Configure o Firebase
Siga os passos abaixo para ativar login e sincronização em nuvem. Gratuito e leva menos de 5 minutos.
1
Crie um projeto Firebase
Acesse console.firebase.google.com → Adicionar projeto → siga o assistente.
2
Registre um app Web
No painel do projeto, clique no ícone </> → dê um nome → clique em Registrar app. Copie os valores do objeto firebaseConfig.
3
Cole as credenciais no arquivo HTML
Abra este arquivo em um editor de texto, encontre window.FIREBASE_CONFIG no início e substitua cada "COLE_AQUI" pelos valores copiados.
4
Ative Authentication e Firestore
Authentication → Começar: ative E-mail/senha e Google.
Firestore Database → Criar banco de dados: escolha Modo de teste.
Plano gratuito Spark do Firebase suporta até 50.000 leituras e 20.000 escritas por dia — mais que suficiente para uso pessoal e acadêmico.
Regra de segurança recomendada após 30 dias: allow read, write: if request.auth != null && request.auth.uid == resource.data.uid;

Uma janela do Google será aberta pedindo autorização — você controla o que permite.

O StudyOS só cria eventos — nunca lê seus dados existentes. O token de acesso fica apenas na memória da sessão.
